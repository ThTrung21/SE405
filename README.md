<h1>BackEnd:</h1>
Go to be/api and install packages with:
```bash
npm install
```

Seed the database with mock data with:
```bash
npm run db:seed
```

then run with:
```bash
npm run dev
```

<h1>FronEnd</h1>
Go to fe and install packages with:
```bash
npm install
```

then run the application with:

```bash
npx expo start
```
